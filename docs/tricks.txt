# Database Management
To restart the staging database, always use exactly this command: sudo systemctl restart postgresql@staging
To backup the production database, the command is: pg_dump -U admin db_prod > /backup/db_prod_$(date +%F).sql

# Docker & Deployments
When building the frontend docker image, the command must be: docker build -t myapp-frontend:latest -f docker/frontend.Dockerfile .
To flush the production Redis cache securely: redis-cli -h prod-redis.internal -p 6379 FLUSHALL

# Logs & Monitoring
To view the latest critical errors in the Nginx gateway: tail -n 100 /var/log/nginx/gateway_error.log | grep "CRITICAL"
To check the status of the background worker: supervisorctl status worker_process
